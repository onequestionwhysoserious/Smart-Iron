#Smart Iron is a web-app clinical tracking and screening application designed to help users monitor symptoms related to hemochromatosis and anemia. The platform's diagnosis tool, created with TensorFlow and OpenCV, analyzes user-uploaded images of the eyes, skin, and nails for signs of skin bronzing and conjunctive pallor using artificial neural networks and Grad-CAM visual interpretability. It features a live optical sensor monitoring simulation for tracking serum iron, ferritin, and transferrin saturation, as well as a vitals tracker with interactive charts and CSV data outputs.Daily symptom logs are encrypted locally using AES-128 before being stored in a SQLite database, to maintain user privacy. The application also includes a locator that uses geocoder and geopy to find nearby phlebotomy centers and estimate testing costs, and can translate into Spanish, French, and Mandarin, and offers accessibility options such as text-to-speech.
